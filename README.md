

# 📊 HR Workforce Attendance Analytics Dashboard

## 🔹 Project Overview

This project is a Power BI dashboard designed to analyze employee attendance patterns, work-from-home (WFH) trends, and sick leave (SL) behavior. It helps organizations monitor workforce productivity and make data-driven HR decisions.

---

## 🔹 Objectives

* Track employee attendance percentage
* Analyze Work From Home (WFH) trends
* Monitor Sick Leave (SL) usage
* Identify patterns over time
* Improve workforce management efficiency

---

## 🔹 Key Features

* 📈 Interactive dashboard with filters (Date, Day of Week)
* 📊 Attendance %, WFH %, and SL % KPIs
* 📅 Time-based trend analysis
* 👨‍💼 Employee-level detailed insights
* 🔍 Dynamic filtering for better analysis

---

## 🔹 Tools & Technologies Used

* Power BI
* Excel (Data Source)
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation

---

## 🔹 Dashboard Insights

* Identified employees with high/low attendance
* Observed consistent WFH patterns across days
* Detected low sick leave percentage overall
* Provided clear visibility of workforce behavior
